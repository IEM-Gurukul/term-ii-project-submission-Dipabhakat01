Problem Statement: Managing library inventory using structured object-oriented principles.
OOP Concepts Used:
Abstraction: Created LibraryItem as an abstract class to define common attributes.
Inheritance: The Book class extends LibraryItem.
Encapsulation: Used private fields and public getters/setters.
Polymorphism: Overrode the getType() method in subclasses.
Collections: Used ArrayList to manage the inventory dynamically.
